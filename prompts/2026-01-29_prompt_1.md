# Task Description
Perform comprehensive spatial transcriptomics analysis of HPV+ head and neck squamous 
cell carcinoma (HNSCC) Slide-TCR-seq data to identify functional TCR-neoantigen pairs 
and characterize the tumor-immune microenvironment. This analysis aims to understand 
immune evasion mechanisms in HPV+ HNSCC and identify potential therapeutic targets 
for CAR-T therapy development.

**Primary Research Questions:**
1. Do mutation-bearing spots co-localize with expanded T cell clones?
2. Are predicted neoantigens expressed in tumor regions where T cells infiltrate?
3. Is there evidence of T cell exclusion from neoantigen-rich areas?
4. Can we identify candidate TCR-neoantigen pairs for therapeutic validation?

**STEP 6: Cluster-Based Annotation Assignment**
This assigns the dominant cell type to each cluster using weighted scoring:
```python
def assign_cluster_based_annotation(adata_obj):
    """
    Assigns cell type to each cluster based on weighted popV predictions.
    Uses min-max normalization and linear weighting within clusters.
    """
    df = adata_obj.obs[['clusters', 'popv_prediction', 'popv_prediction_score']].copy()
    df['popv_prediction_score'] = pd.to_numeric(df['popv_prediction_score'], errors='coerce')
    
    # Min-Max normalization within each cluster
    def min_max_normalize(x):
        x_min, x_max = x.min(), x.max()
        if x_max - x_min == 0:
            return np.zeros(len(x))
        return (x - x_min) / (x_max - x_min)
    
    df['normalized_score'] = df.groupby('clusters')['popv_prediction_score'].transform(min_max_normalize)
    
    # Linear weighting (normalize to sum to 1 within each cluster)
    def linear_weights(x):
        total = x.sum()
        if total == 0:
            return np.ones(len(x)) / len(x)
        return x / total
    
    df['weight'] = df.groupby('clusters')['normalized_score'].transform(linear_weights)
    df['weighted_score'] = df['popv_prediction_score'] * df['weight']
    
    # Aggregate by cluster and cell type, find dominant type
    cluster_type_scores = df.groupby(['clusters', 'popv_prediction'])['weighted_score'].sum().reset_index()
    dominant_types = (
        cluster_type_scores
        .sort_values('weighted_score', ascending=False)
        .drop_duplicates('clusters')
        .set_index('clusters')['popv_prediction']
    )
    
    # Map dominant type back to all cells
    adata_obj.obs['final_annotation'] = adata_obj.obs['clusters'].map(dominant_types)
    
    return adata_obj

# Apply cluster-based annotation
adata_pp = assign_cluster_based_annotation(adata_pp)
```

For mutations NOT found in TSNAdb/NEPdb, predict HLA binding de novo.

**APPROACH:**
1. Generate mutant peptides (8-11mers)
   - Center mutation in peptide
   - Generate all possible peptide lengths
2. Get patient HLA type (if available)
   - Or use common HLA alleles for HNSCC population
3. Run NetMHCpan prediction:
   ```bash
   netMHCpan -p peptides.txt -a HLA-A02:01,HLA-A01:01 -l 9 > binding_predictions.txt
   ```

4. Filter strong binders:
   - %Rank < 0.5 = strong binder

# Context

## Scientific Background
These are Slide-TCR-seq spatial transcriptomics samples from HPV16+ HNSCC tumors 
provided by Dr. Sophia Liu (Ragon Institute). The data captures both gene expression 
and TCR sequences with spatial resolution. The central hypothesis is that APOBEC3B-driven 
mutagenesis creates neoantigens that should be recognized by tumor-infiltrating CD8+ 
T cells, yet these tumors maintain immune evasion. Understanding the spatial 
relationships between mutated cells, viral infection status, and T cell infiltration 
may reveal mechanisms of immune escape.

## Technical Constraints
- Working with ~30K cells per puck after QC filtering
- Read depth may limit single-cell mutation calling - aggregate by cell type first
- Use spatial coordinates from original Slide-seq bead positions
- BAM files contain cell barcodes in XB tag (matched files) or XC tag (all_illumina)
- Reference genome: GRCh38 with GENCODE v49 annotations

## Dependencies Between Tasks
1. Cell type annotation must complete before SComatic (need cell type labels)
2. TCR calling can run in parallel with cell type annotation
3. HPV detection can run in parallel with mutation detection
4. Neoantigen prediction requires mutation calling output
5. Final integration requires all Phase 1-3 outputs
6. Publication figures require all analyses complete

## Success Criteria
- Cell type annotation achieves >80% confidence scores
- Mutation detection identifies variants in tumor regions
- HPV+ cells detected and mapped spatially
- CD8+ T cells and TLS regions clearly delineated
- At least 3 candidate TCR-neoantigen pairs identified for validation
- Publication-ready figures generated
